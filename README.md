#### 架构

#### 安装

1. npm 安装


#### 介绍

- 安装npm  
npm install
- 运行服务  
npm run serve
- 执行编译  
npm run build
- 执行测试  
npm run test
- 执行检查
npm run lint


#### 参与mecm-fe贡献

1. fork mecm-fe的个人分支
2. 提交代码到个人分支
3. 创建个人分支到主线分支的PR
4. 由仓库committer审核合入
